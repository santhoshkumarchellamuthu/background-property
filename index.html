BACKGROUND PROPERTY

background-color

The background-color property sets the background color of an element. The value can be any valid CSS color value, such as red, #ff0000, or rgb(255, 0, 0).

background-color:red;

This text will have a red background.

background-image

The background-image CSS property sets one or more background images on an element.

background-image:url("");

background-image:url(""),
url("");

background-attachment

The background-attachment CSS property sets whether a background image's position is fixed within the viewport, or scrolls with its containing block

background-attachment:repeat;

background-attachment:repeat-x;

background-attachment:repeat-y;

background-attachment:space;

background-attachment:round;

background-attachment:no-repeat;

background-attachment

The background-attachment CSS property sets whether a background image's position is fixed within the viewport, or scrolls with its containing block

background-attachment:scroll;

this text will be the background-attachment

background-attachment:fixed;

this text will be the background-attachment

background-attachment:local;

this text will be the background-attachment

background-position

The background-position CSS property sets the initial position for each background image. The position is relative to the position layer set by background-origin.

background-position:top;

background-position:left;

background-position:center;

background-origin

The background-origin CSS property sets the background's origin: from the border start, inside the border, or inside the padding.

background-origin:border-box;

background-origin:padding-box;

background-origin:content-box;

background-size

The background-size CSS property sets the size of the element's background image. The image can be left to its natural size, stretched, or constrained to fit the available space

background-size:contain;

background-size:cover;

background-size:50%;
